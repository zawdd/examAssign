输入限制：

要求课程只能为 
马克思主义基本原理
毛泽东思想和中国特色社会主义理论体系概论
思想道德修养与法律基础
中国近现代史纲要

所能处理的校区为
南校区 东校区 珠海校区

输入中不能有北校区，或其他课程字样，否则程序会异常
输入文件格式必须为xls后缀，且必须放在第一个sheet里面


输出结果：
东校区输出到eastRESULT.xls
南校区输出到southRESULT.xls
珠海校区输出到zhuhaiRESULT.xls

每个文件中包含四种课程，若为空表示该校区本次没有该课程的考试
南校区分为1,2,3三种分配方案，有的方案可能无法得出结果，请人工选取可以使用的
第1,2种方案仅仅利用逸夫楼，第3种方案利用逸夫楼和一教

使用说明：
按照格式整理好要输入的考试信息文件
运行程序时不能打开*RESULT.xls文件，否则会出错
程序利用到根目录下的rooms.xls文件，请勿编辑此文件

输入文件包含以下信息，每个一列
教学班ID	课程名称	教师名称	教学班人数	校区
